============
ddr-explorer
============

**ddr-explorer** is a REST back-end for Brian Yamazaki's app.


.. REQUIREMENTS
.. ============
.. 
.. * Python 3.5
.. * Django 2.*
.. 
.. 
.. INSTALL
.. =======
.. 
.. If you have downloaded the source code:
.. 
.. 	   sudo make install
.. 	
.. A distribution package can be obtained for manual installation at:
.. 
..     URL
.. 
.. 
.. SOURCE
.. ======
.. 
.. ddr-explorer's git repo is available on GitHub, which can be browsed at:
.. 
..     https://github.com/densho/ddr-explorer
.. 
.. and cloned using:
.. 
..     git clone git://github.com/densho/ddr-explorer.git ddr-explorer
